How USD returns are calculated

CEDEARs & Arg Stocks: USD value = ARS price / MEP rate. Entry cost = purchase ARS price / MEP rate at purchase. This reflects the actual dollars you'd get via the MEP bond operation.

US Stocks: Already priced in USD. No FX conversion needed.

Return Decomposition: Capital return = asset price change in native currency. Currency return = MEP rate change since purchase. Total USD return = (1 + capital) × (1 + currency) - 1.

MEP rate source: AL30 bond pair via Data912. CCL rate used for displaying US stocks in ARS.
